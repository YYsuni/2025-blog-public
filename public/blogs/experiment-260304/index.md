# 26-03-04

进行了第一次实验

原理就是银纳米簇会放光，类似ThT，然后T链能猝灭，但是HPV16的结合能力比T强，能把T挤下去重新发光。
## Process
1. 合成银纳米簇
2. 取6μL 100μM的Ag1链，95°C 5分钟，然后25°C下孵育30分钟
3. 孵育过程中配置0.2mM的硝酸银溶液（用水稀释）
4. 孵育结束后Ag1加10μLPB和54μLH₂O，然后加入20μLAgNO₃，4°C下20分钟
5. 十分钟后配置硼氢化钠（用水稀释），这个有效时间不超过十分钟，等那二十分钟一到马上加10μL进去，然后振动1min，避光保存6h
6. 要到6h时，处理T和H16，T的浓度跟银（银纳米簇，后面都喊银）相等，H最大的跟银相等，设置梯度变小，用去离子水稀释
7. 按组进行添加，分组如下（按buffer——最后添加到200μL的buffer）

8. 37°C孵育一个半小时，测光，参数如下

	吸收 500 - 650
	狭缝 5/5
	电压 700V

中间存在问题，可能问题有：
1. 硝酸银溶液需要重新配置
2. 在避光6h后，先摇匀5-10s再使用

提出了两个新方案
方案2和3（之前的方案自动成为方案1）
PS：与方案1的差异主要是在合成银纳米簇阶段，后面步骤一模一样
- 方案2（Ag-DNA=45μM 100μL）
	15μL 100μM DNA + 18μL 1mM AgNO₃+49μL 20mM pH=6.6 PB
	4℃下 20min 后 + 18μL 1mM NaBH<sub>4</sub>
	极速mix，避光18h.
- 方案3（15μM 45μL）
	6μL 100μM DNA + 12μL H<sub>2</sub>O +4μL 20mM pH=6.6 Tris - HNO<sub>3</sub> + 9μL 0.4mM AgNO<sub>3</sub>
	4℃ 15min + 9μL 0.4mM NaBH<sub>4</sub> 极速mix
	4℃下避光12h.
