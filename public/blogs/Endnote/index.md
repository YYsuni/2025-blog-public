# 一、复旦大学图书馆

1. 在浏览器输入复旦大学图书馆并进入

2. 点击数据库→常用数据库→Endnote

   ![1](https://img.charrider.dpdns.org/file/1773238501572_657d9520-c399-4c33-a1bd-eca73be63aca_页面_006.jpg)

3. 向下滑动，点击网上办事服务大厅 

   ![](https://img.charrider.dpdns.org/file/1773238588288_657d9520-c399-4c33-a1bd-eca73be63aca_页面_008.jpg)

4. 点击右下角查看全部 

   ![](https://img.charrider.dpdns.org/file/1773238646485_657d9520-c399-4c33-a1bd-eca73be63aca_页面_009.jpg)

5. 往下划，找到EndNote文献管理工具，点击进入服务 

   ![](https://img.charrider.dpdns.org/file/1773238694424_657d9520-c399-4c33-a1bd-eca73be63aca_页面_010.jpg)

6. 就来到了下载地点，即[EndNote下载](https://library.fudan.edu.cn/e8/b0/c42801a518320/page.psp)，按照官方页面教程来即可

# 二、中国农业科技文献与信息服务平台

1. 浏览器搜索中国农业科技文献与信息服务平台
2. 在常用资源里面点击Endnote即来到了[下载网站](https://www.nais.net.cn/publish/default/fwgg/content/2024091109384415126.html)
